# 📊 Customer Churn Analysis (Excel)

## 📌 Overview

This project analyzes customer churn data using Excel to identify patterns and reduce customer loss.

## 🎯 Objectives

* Understand churn behavior
  
* Identify key factors
  
* Build a simple dashboard

## 🛠 Tools

* Microsoft Excel
  
* Pivot Tables
  
* Charts

## 📊 Key Insights

* High churn in month-to-month customers
  
* Lack of tech support increases churn
  
* New customers churn more
  
* Electronic payment users have higher churn

## 💡 Recommendations

* Promote long-term plans
  
* Improve customer support
  
* Target new customers
  
* Encourage digital payments

## 🚀 Conclusion

The project helps in understanding customer behavior and improving retention strategies.
